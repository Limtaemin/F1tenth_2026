# 01. 설치와 빌드

이 문서는 F1TENTH 레포를 처음 받은 뒤 빌드하고 실행 준비를 하는 절차를 설명한다.

---

## 1. 전제 조건

기본적으로 아래 환경을 가정한다.

| 항목 | 기준 |
|---|---|
| OS | Ubuntu 20.04 |
| ROS2 | Foxy |
| 빌드 도구 | colcon |
| 언어 | Python3, C++ |

작업 경로는 아래를 기준으로 설명한다.

```bash
~/race_stack/f1tenth
```

---

## 2. 레포 위치로 이동

```bash
cd ~/race_stack/f1tenth
git branch
git status
```

정상 예시:

```text
On branch korean-docs-structure
Your branch is up to date with 'origin/korean-docs-structure'.

nothing to commit, working tree clean
```

---

## 3. ROS2 환경 적용

터미널을 새로 열었다면 먼저 ROS2 환경을 적용한다.

```bash
source /opt/ros/foxy/setup.bash
```

빌드 후에는 이 레포의 install 환경도 적용한다.

```bash
source install/setup.bash
```

---

## 4. 의존성 설치

```bash
sudo apt update
sudo apt install -y \
  python3-colcon-common-extensions \
  python3-rosdep \
  python3-vcstool \
  python3-pip
```

`rosdep` 초기화가 안 되어 있다면 아래를 실행한다.

```bash
sudo rosdep init || true
rosdep update
```

레포 기준 의존성을 설치한다.

```bash
cd ~/race_stack/f1tenth
rosdep install --from-paths src --ignore-src -r -y
```

---

## 5. serial_driver 설치

`vesc_driver` 빌드 중 아래 오류가 날 수 있다.

```text
fatal error: serial_driver/serial_driver.hpp: No such file or directory
```

이 경우 ROS2 Foxy용 serial driver를 설치한다.

```bash
sudo apt update
sudo apt install -y ros-foxy-serial-driver
```

---

## 6. 빌드

스크립트 사용:

```bash
cd ~/race_stack/f1tenth
bash scripts/01_build.sh
```

직접 빌드:

```bash
cd ~/race_stack/f1tenth
colcon build --symlink-install
```

빌드가 끝나면 환경을 적용한다.

```bash
source install/setup.bash
```

---

## 7. 빌드 성공 확인

```bash
ros2 pkg list | grep -E "vesc|f1tenth|particle|pure|stanley|safety"
```

차량 bringup 이후에는 토픽을 확인한다.

```bash
ros2 topic list
```

---

## 8. vesc_driver 빌드 오류 메모

이 브랜치에서는 `vesc_driver`가 `serial_driver` 라이브러리를 제대로 찾고 링크하도록 `CMakeLists.txt`를 수정했다.

확인 파일:

```text
src/f1tenth_system/vesc/vesc_driver/CMakeLists.txt
```

핵심 확인 명령:

```bash
grep -n "serial_driver" src/f1tenth_system/vesc/vesc_driver/CMakeLists.txt
```

## 10. 자주 쓰는 확인 명령

| 목적 | 명령 |
|---|---|
| 토픽 목록 | `ros2 topic list` |
| LiDAR 확인 | `ros2 topic echo /scan` |
| odom 확인 | `ros2 topic echo /odom` |
| TF 확인 | `ros2 run tf2_tools view_frames` |
| 노드 확인 | `ros2 node list` |
| launch 파일 찾기 | `find src -name "*.launch.py" | sort` |
| config 파일 찾기 | `find src -name "*.yaml" -o -name "*.yml" | sort` |
