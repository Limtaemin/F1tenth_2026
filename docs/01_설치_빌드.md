# 01. 설치 및 빌드 가이드

이 문서는 F1TENTH 레포를 처음 받은 뒤 빌드하는 방법을 설명한다.

기준 환경:

```text
Ubuntu 20.04
ROS2 Foxy
colcon
rosdep
1. ROS2 환경 적용

터미널을 새로 열 때마다 ROS2 환경을 적용해야 한다.

source /opt/ros/foxy/setup.bash

확인:

echo $ROS_DISTRO

정상이라면:

foxy

가 출력된다.

2. 레포 위치로 이동
cd ~/race_stack/f1tenth

현재 위치 확인:

pwd

예상 결과:

/home/lim/race_stack/f1tenth
3. 의존성 설치

기본 의존성 설치:

sudo apt update
sudo apt install python3-colcon-common-extensions python3-rosdep -y

rosdep 초기화가 안 되어 있다면:

sudo rosdep init
rosdep update

이미 초기화되어 있다는 메시지가 나오면 rosdep update만 실행한다.

rosdep update

레포 의존성 설치:

rosdep install --from-paths src --ignore-src -r -y
4. serial_driver 의존성

vesc_driver 빌드 중 아래 에러가 날 수 있다.

fatal error: serial_driver/serial_driver.hpp: No such file or directory

이 경우 다음 패키지를 설치한다.

sudo apt install ros-foxy-serial-driver -y
5. 빌드

일반 빌드:

colcon build --symlink-install

또는 helper script 사용:

bash scripts/01_build.sh

성공하면 마지막에 failed가 없어야 한다.

예상 예시:

Summary: 14 packages finished

중요한 것은 다음 문구가 없어야 한다.

failed
aborted
6. 빌드 후 환경 적용

빌드가 끝나면 반드시 workspace 환경을 적용한다.

source install/setup.bash

패키지 확인:

ros2 pkg list | grep vesc

예상 출력:

vesc_ackermann
vesc_driver
vesc_msgs
7. 특정 패키지만 다시 빌드하기

특정 패키지만 다시 빌드하려면 다음처럼 한다.

colcon build --symlink-install --packages-select vesc_driver

예를 들어 vesc_driver만 고친 경우:

rm -rf build/vesc_driver install/vesc_driver log
colcon build --symlink-install --packages-select vesc_driver
8. 자주 나오는 오류
8.1 serial_driver.hpp 없음

에러:

serial_driver/serial_driver.hpp: No such file or directory

해결:

sudo apt install ros-foxy-serial-driver -y
8.2 undefined reference to drivers::serial_driver

에러:

undefined reference to drivers::serial_driver::SerialPort::open()
undefined reference to drivers::serial_driver::SerialPort::async_send()

원인:

serial_driver 헤더는 찾았지만 라이브러리 링크가 제대로 안 된 상태

해결:

src/f1tenth_system/vesc/vesc_driver/CMakeLists.txt에 serial_driver 의존성을 추가해야 한다.

핵심 내용:

find_package(serial_driver REQUIRED)

ament_target_dependencies(${PROJECT_NAME}
  serial_driver
)
9. 빌드 완료 확인 명령어

아래 명령으로 한 번에 확인할 수 있다.

cd ~/race_stack/f1tenth

echo "=== ROS_DISTRO ==="
echo $ROS_DISTRO

echo
echo "=== 패키지 확인 ==="
ros2 pkg list | grep -E "vesc|pure_pursuit|particle_filter|stanley"

echo
echo "=== 전체 빌드 ==="
colcon build --symlink-install
10. 다음 단계

빌드가 성공했다면 다음 문서로 넘어간다.

docs/00_전체_개요.md
docs/07_config_설명.md

실차를 켜기 전에는 반드시 bringup과 센서 토픽을 확인한다.

bash scripts/02_bringup.sh
ros2 topic list
